# 🧮 Scientific Calculator

A modern and responsive **Scientific Calculator** built using **HTML, CSS, and JavaScript**. It performs both basic and advanced mathematical calculations with a clean interface and smooth user experience. The calculator works entirely in the browser without requiring any installation or external libraries.

## ✨ Features

* ➕ Basic arithmetic operations
* 📐 Scientific functions
* √ Square Root
* x² and Power calculations
* 📊 Logarithmic functions
* 📏 Trigonometric functions (Sin, Cos, Tan)
* π (Pi) and other mathematical constants
* 🧹 Clear and Delete buttons
* ⌨️ Responsive and user-friendly interface
* 📱 Mobile-friendly design
* ⚡ Fast calculations using JavaScript
